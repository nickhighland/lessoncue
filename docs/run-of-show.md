# Lesson planning and run-of-show

LessonCue keeps the plan, operator guidance, and live timing on the self-hosted server. Notes and private monitoring are shown only to signed-in planners and controllers; they are never sent to an audience display manifest.

## Build the plan

Open **Classes**, choose a lesson, and arrange its cues in playback order. Each cue can have **Teacher / volunteer notes** and can be marked **Flexible timing** when it may be shortened or omitted if the lesson runs long. The estimated run time accounts for trims, still-image duration, playback speed, and repeat count. Media without a known duration is called out and excluded from the estimate.

Use **Substitute or teacher instructions** for room setup, handoffs, accessibility details, or other whole-lesson guidance. Optionally enter an HTTP or HTTPS **pre-roll livestream monitor** for a private confidence camera or stream page. Embedded sites can still refuse to appear in an iframe; the controller always provides a direct new-tab link.

LessonCue warns when two estimated lesson windows overlap in the same classroom. It does not treat simultaneous lessons in different rooms as a conflict.

## Print a run sheet

Choose **Print run sheet** on the lesson page. The print layout includes:

- lesson, classroom, date, and designated start;
- whole-lesson substitute instructions;
- every pre-roll, countdown, and main cue in order;
- estimated cue times and durations;
- flexible-time markers and cue notes; and
- estimated finish time.

Only the run sheet is printed; navigation and dialog controls are removed by the print stylesheet.

## Copy or move a lesson

Choose **Copy / move** on a lesson. Select the destination classroom, date, and title.

- **Copy** retains the original and creates a complete independent lesson, including media references, pre-roll/countdown roles, trims, fades, playback controls, notes, flexible markers, and private monitor URL.
- **Move** changes the existing lesson and shifts its availability, pre-roll, designated-start, and expiry timestamps by the same number of days.

Temporary media retention is extended when the destination date requires it. Moving a lesson generated by a recurring schedule detaches it from that schedule so later generation does not overwrite the operator's change.

## Calendar views

The **Calendar** page offers Agenda, Day, Week, Month, and Room views. Use the date picker, previous/next controls, or **Today** to navigate. Every scheduled lesson shows its estimated duration and finish; an orange conflict marker identifies same-room overlaps.

## Run the lesson from a phone

The class or universal controller shows the current cue's notes, whole-lesson substitute instructions, total time remaining, and estimated finish. A warning appears when the live estimate runs beyond the planned finish. Flexible cues are labeled in the media list so an authorized operator can make an informed adjustment.

During the configured pre-roll window, the optional private monitor opens automatically on the controller. It can also be opened or hidden manually. The monitor URL is never included in Android TV, Apple TV, or browser-player manifests.
