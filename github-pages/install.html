HEADLESS LINUX INSTALLATION

Install entirely over SSH.

These commands install LessonCue on a 64-bit Ubuntu or Debian server with no desktop or GUI. Use a non-root SSH account with sudo.

Keep the server on your trusted local network. Do not forward TCP port 8080 from the public internet.
STEP 1

Connect to the server

ssh YOUR_USERNAME@SERVER_IP
STEP 2 · COPY THESE COMMANDS

Download and install the latest release

sudo apt-get update
sudo apt-get install -y curl ca-certificates
curl -fsSL https://raw.githubusercontent.com/nickhighland/lessoncue/main/installers/linux/install-latest.sh | bash

The installer waits for the service and prints LessonCue is ready with http://lessoncue.local:8080 and a numeric fallback.

STEP 3

Open the local interface

Open http://lessoncue.local:8080 from another computer on the same network. Create your local administrator account, then use the random pairing PIN shown on the Dashboard to pair a TV. Administrators can change the .local name in Settings.

STEP 4

Verify and find the local address

sudo systemctl status lessoncue --no-pager
curl -fsS http://127.0.0.1:8080/health && echo
hostname -I | awk '{print "http://" $1 ":8080"}'

The complete browser interface, API, database, schedules, and media are served by this local machine. No hosted interface is used.

CELLPHONE CONTROLLER

Run playback from the room

http://lessoncue.local:8080/controller
http://SERVER-IP:8080/controller

Open the address on the same trusted Wi-Fi, sign in, and choose a screen and lesson. Use Safari's Add to Home Screen or the Android browser's Add to Home screen / Install app action to save it like an app.

LOGS AND RECOVERY

Useful SSH commands

sudo journalctl -u lessoncue -f
sudo systemctl restart lessoncue
sudo systemctl status lessoncue --no-pager

Press Ctrl+C to stop following logs. Disconnecting SSH does not stop LessonCue.

PASSWORD RECOVERY

Reset an administrator over SSH

sudo -u lessoncue env LESSONCUE_DATA_PATH=/var/lib/lessoncue /opt/lessoncue/LessonCue.Server --list-admins

sudo -u lessoncue env LESSONCUE_DATA_PATH=/var/lib/lessoncue /opt/lessoncue/LessonCue.Server --reset-password YOUR_USERNAME

The new password is entered invisibly at the prompt, stored as a secure hash, and existing sessions for that account are signed out.
