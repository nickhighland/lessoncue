PRODUCT ROADMAP

Make every room more ready.

The roadmap keeps LessonCue local-first and easy to operate while strengthening live playback, planning, accessibility, security, recovery, and large-site administration.

Fifteen recommended priorities are complete through v0.24.0. Administrators now define the approved media folder hierarchy and tags that uploaders can choose in lessons and the Media Library. The comprehensive checkbox roadmap remains the source of truth for shipped and remaining work.
SHIPPED · v0.24

Consistent media organization

Define approved hierarchical folders and tags once, then give every uploader clear choices in lesson and Media Library workflows.

SHIPPED · v0.22

Edit safely and recover deletions

Edit or remove classrooms and recover deleted classrooms, lessons, and media for 30 days from an administrator-only recycling bin.

SHIPPED · v0.21

Dedicated classroom controllers

Give each room a branded path or hostname, room/lesson QR launch, expiring restricted sessions, and a saved-to-phone controller while retaining a PIN-protected universal remote.

SHIPPED · v0.20

Optional secure remote access

Connect an administrator-owned hostname through a restricted, outbound-only Cloudflare Tunnel while keeping local-only operation as the default.

SHIPPED · v0.19

Bulk planning and media operations

Upload multiple files and select lessons, playlist cues, or media to rename, move, shift, archive, organize, retain, or remove them together.

SHIPPED · v0.18

Actionable screen diagnostics

Inspect cache files, queued downloads, decoders, recent errors, network quality, and clock drift—with privacy-gated one-time screenshots.

SHIPPED · v0.12–0.17

Edit, reuse, and organize

Visual timelines, media folders and versions, presentation conversion, recurring templates, restore, reliable TV video, and granular permissions are complete.

RELIABILITY

Prove physical-device workflows

Expand the automated browser suite into repeatable Android TV, Fire TV, and Apple TV hardware matrices for offline recovery, upgrades, and live control.

CONTROL

A clearer room controller

Add live push state, blackout, volume, command confirmation, lock mode, haptic feedback, screen groups, operator handoff, and simplified volunteer controls.

ACCESSIBILITY

Usable by more people

Complete WCAG 2.2 AA review, captions and transcripts, localization, high-contrast themes, reduced motion, large-touch modes, and screen-reader coverage.

SECURITY

Scale permissions safely

Add granular roles, passkeys and TOTP, active-session control, optional directory sign-in, signed updates, audit export, and stronger governance options.

FLEET & INTEGRATIONS

Support larger installations

Add device profiles, staged updates, screen groups, webhooks, service tokens, monitoring, calendar and LMS connectors, and production-system integrations.
